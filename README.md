# Magi Boutique

Este projeto é um site simples desenvolvido para a **Magi Boutique**, com o objetivo de centralizar os principais links e formas de contato da loja em uma única página.

## 🚀 Funcionalidades

* Acesso rápido ao Instagram
* Contato direto via WhatsApp
* Link para grupo no WhatsApp
* Localização da loja física
* Layout responsivo (mobile e desktop)

## 💻 Tecnologias utilizadas

* HTML5
* CSS3

## 🎯 Objetivo

Facilitar o acesso dos clientes às redes sociais e canais de atendimento da Magi Boutique, oferecendo uma experiência simples, rápida e visualmente agradável.
